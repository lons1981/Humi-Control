# Humi-Control
Der moderne, vollautomatische Klima-Controller für Zigarrenhumidore. Misst und steuert zuverlässig Temperatur und Luftfeuchtigkeit für optimale Lagerbedingungen. Mit farbigem Display, automatischer Befeuchtung, WLAN-Anbindung und Home-Assistant-Support. Einfache Bedienung, OTA-Updates und modulare Erweiterbarkeit. Perfektes Klima – jederzeit.
